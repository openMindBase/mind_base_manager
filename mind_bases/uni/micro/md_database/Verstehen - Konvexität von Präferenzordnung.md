## Description

- Schwache Konvexität: Ist $A \sim B$ und $C$ deren Mischung, so gilt $C \succsim A$ (also auch $C \succsim B$).
- Starke Konvexität: Ist $A \sim B$ und $C$ deren Mischung, so gilt $C \succ A$ (also auch $C \succ B$).
